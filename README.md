# HOSTEL-HYGIENE-REPORT-SYSTEM
Simple Python project for hostel hygiene reporting.

##Project overview:

      -The Hostel Hygiene Report System is a simple Python-based console application that allows hostel students to submit hygiene-related reports.
      -Students can report issues regarding water quality, food quality, cleanliness, kitchen hygiene, waste management, and general complaints/feedback.
      -The system collects:
             -Student details
             -Hostel block
             -Room number
             -Hygiene ratings
             -Feedback/complaints
    -Finally, it generates a complete hygiene report summarizing the student's inputs.
##Features: 

      -Student information input (name, registration number, gender).
      -Dynamic hostel block selection (different for boys and girls).
      -Room number entry.
      -Multiple hygiene categories.
      -User-friendly menu for reporting.
      -Ratings: Good / Average / Poor.
      -Option to give detailed feedback.
      -Final summary report generated in a clean format.
##Technologies / Tools Used:

     -Programming Language: Python
     -IDE/Editor:VS Code
     -Version Control: Git / GitHub
##Steps to Install and Run the Project

    1. Install Python
         -Make sure Python(3.x) is installed on your computer.
    2.  Download the Project
         -by this link( https://github.com/raushansingh2602/HOSTEL-HYGENE-REPORT-SYSTEM.git )
    3.  Open the Project Folder
          -check whether this project file located (hostel_hygiene_report.py)
    4.  Run the Python Script
          - Run the project using Python
    5.  Enter Student Details
          -Program will ask for
             -Student Name
             -Registration No.
             -Gender(Boy/Girl)
    6.  Select Hostel Block
          -Based on yopur Gender,choose your hostel block from the list displayed.
          -Enter the number corresponding to your block.
    7.  Enter Room Number
          - Input your room number (example A001)
    8.  Submit Hygiene Reports
          - Choose a category in which you want to report:
                 a.Water Quality
                 b.Food Quality
                 c.Cleanliness
                 d.Kitchen Hygiene
                 e.Waste Management
                 f.Complaints/Feedback
                 g.Finish Reporting
          -For Option (a-g)
              - Enter rating (Good/Average/Poor)
              - Enter Feedback (or type None)
          -For Option (6)
              - Enter any general complaint /Feedback
    9.  Finish Reporting 
          -Choose Option 7(finish)
          -Program will generate the Final Hygiene Report on the screen:
              -Student Details
              -Hostel block & room no.
              -Rating and Feedback for each category
    10.  Done
          -Your Hygiene report is complete!
